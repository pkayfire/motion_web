## Getting Started

Install the module with: `npm install motion-web`

```js
var motion-web = require('motion-web');
motion-web.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g motion-web
$ motion-web --help
$ motion-web --version
```


```sh
# creates a browser.js
$ grunt browserify
```
